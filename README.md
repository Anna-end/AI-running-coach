# 🏃 AI Running Coach
Проект в разработке

Веб-приложение, где AI-агент составляет персональные планы тренировок по бегу на основе анкеты. Пользователь отмечает выполненные тренировки, вносит правки и смотрит прогресс.

## 🧰 Стек
Next.js — фронт и API routes

React Hook Form + Zod — валидация формы

TanStack Query — кеширование и синхронизация данных

Supabase — БД, авторизация, хранение

Vercel — деплой


## ✅ Функционал
📝 Форма для заполнения (уровень, цель, дни, ограничения)

🤖 Генерация плана AI-агентом

✅ Отметка тренировок как выполненных

✏️ Редактирование тренировок

📊 Просмотр результатов и прогресса
